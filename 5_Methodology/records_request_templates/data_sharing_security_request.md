Dear Public Records Officer:

Pursuant to Florida's Public Records Law, Chapter 119, Florida Statutes, I am requesting the following records related to ALPR data sharing arrangements and security practices:

### Data Sharing Agreements

1. **Inter-Agency Sharing Agreements** - All agreements, MOUs, or contracts governing ALPR data sharing with:
   - Other Florida law enforcement agencies
   - Federal agencies (FBI, ICE, CBP, DHS, etc.)
   - Out-of-state agencies
   - Private entities or contractors

2. **Flock Safety Network Participation Agreement** - Any agreement governing:
   - Access to other agencies' ALPR networks
   - Other agencies' access to OCSO data
   - Terms and conditions of network participation
   - Data use restrictions

3. **Agency Access List** - Current list of all agencies with access to OCSO ALPR data, including:
   - Agency name
   - Date access granted
   - Scope of access (real-time, historical, alerts only, etc.)

4. **Access Request/Approval Records** - Documentation of:
   - Requests from other agencies for ALPR data
   - OCSO decisions on data sharing requests
   - Any denied requests and reasons

### Security Records

5. **Security Incident Reports** - Any records of:
   - Data breaches involving ALPR data
   - Unauthorized access attempts
   - Security vulnerabilities identified
   - Corrective actions taken

6. **Security Assessments** - Any:
   - Penetration testing reports
   - Vulnerability assessments
   - Third-party security audits
   - Compliance certifications

7. **Breach Notification Records** - Any:
   - Notifications sent to affected individuals
   - Notifications to regulatory agencies
   - Internal breach response documentation

8. **Flock Safety Security Documentation** - Any security-related documents received from Flock Safety, including:
   - SOC 2 reports or certifications
   - Security architecture documentation
   - Incident response procedures
   - Data encryption specifications

### Access Control Records

9. **User Access Logs** - Records showing:
   - Which personnel have ALPR system access
   - Access level/permissions for each user
   - Access grants and revocations
   - Login/authentication logs (aggregated statistics acceptable)

10. **Access Audit Records** - Any audits of:
    - Appropriate access levels
    - Terminated employee access removal
    - Suspicious access patterns

### Third-Party Access

11. **Flock Safety Access Records** - Documentation of:
    - Flock Safety employee access to OCSO data
    - Purpose and scope of vendor access
    - Access logging and monitoring

12. **Subcontractor Information** - Per MSA Section 2.1, Flock may use third parties. Records of:
    - Any third-party subcontractors with data access
    - Subcontractor agreements or certifications
    - Data handling requirements for subcontractors

### Preferred Format

Please provide records in electronic format (PDF preferred) to: [email address]

### Exemption Acknowledgment

I acknowledge that some security details may be exempt from disclosure. If claiming exemptions, please:
1. Cite specific statutory basis
2. Provide non-exempt portions of responsive records
3. Describe generally the nature of withheld information

### Response Timeline

Per Florida Statutes Section 119.07, I expect a response within a reasonable time.

Thank you for your assistance.

Sincerely,

[Your Name]

## Notes for Requestor

- Security assessment details may be partially exempt
- Breach records should be largely disclosable after incident resolution
- Inter-agency agreements are generally public records
- User lists may have some privacy considerations
- Focus on policies and aggregate data if specific details denied
